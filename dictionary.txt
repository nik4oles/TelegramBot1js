class - базовая единица структуры программы, описывающая объект
object - экземпляр класса, имеющий свои данные и поведение
method - функция, определенная внутри класса, выполняющая действия
variable - именованное хранилище данных
inheritance - механизм наследования свойств и методов от родительского класса
polymorphism - способность объектов вести себя по-разному в зависимости от контекста
encapsulation - сокрытие внутренних данных объекта и предоставление доступа через методы
abstraction - представление сущности без деталей реализации
interface - набор методов, который класс должен реализовать
exception - событие, нарушающее нормальный ход выполнения программы
thread - поток исполнения, позволяющий выполнять несколько задач одновременно
JVM - виртуальная машина Java, исполняющая байт-код
JDK - набор инструментов для разработки на Java
JRE - среда выполнения Java
bytecode - промежуточный код, получаемый из исходного кода Java
garbage collection - автоматическое освобождение памяти от неиспользуемых объектов
annotation - метаданные, предоставляющие информацию о коде
reflection - механизм доступа к информации о классах и объектах во время выполнения
generics - обобщенное программирование, позволяющее работать с различными типами данных
lambda - анонимная функция, используемая для краткого описания операций
stream - поток данных, позволяющий обрабатывать элементы коллекции
API - набор интерфейсов для взаимодействия с библиотекой или системой
library - набор классов и функций, предоставляющих определенные возможности
framework - каркас для разработки приложений, предоставляющий структуру и компоненты
Spring - популярный фреймворк для разработки Java-приложений
Hibernate - фреймворк для работы с базами данных
Maven - инструмент для управления сборкой проектов
Gradle - еще один инструмент для управления сборкой проектов
JUnit - фреймворк для unit-тестирования
Mockito - фреймворк для mock-тестирования
Lombok - библиотека для уменьшения шаблонного кода
IntelliJ IDEA - популярная IDE для разработки на Java
Eclipse - еще одна популярная IDE для Java
NetBeans - еще одна IDE для Java
Git - система контроля версий
GitHub - платформа для хостинга Git-репозиториев
Agile - методология разработки ПО, основанная на гибкости и адаптивности
Scrum - фреймворк управления проектами в Agile
Kanban - метод визуализации и управления рабочим процессом
protocol - набор правил для обмена данными между устройствами
TCP - протокол управления передачей, обеспечивающий надежную доставку данных
UDP - протокол пользовательских датаграмм, обеспечивающий быструю, но ненадежную доставку
IP - интернет-протокол, определяющий адресацию устройств в сети
HTTP - протокол передачи гипертекста, используемый для доступа к веб-сайтам
HTTPS - безопасная версия HTTP, использующая шифрование
FTP - протокол передачи файлов
SMTP - протокол передачи почты
DNS - система доменных имен, преобразующая имена доменов в IP-адреса
firewall - межсетевой экран, защищающий сеть от несанкционированного доступа
router - маршрутизатор, направляющий пакеты данных между сетями
switch - коммутатор, соединяющий устройства в одной сети
subnet - подсеть, часть сети, имеющая свой диапазон IP-адресов
VLAN - виртуальная локальная сеть, логическое разделение сети
VPN - виртуальная частная сеть, обеспечивающая безопасное соединение
NAT - преобразование сетевых адресов, позволяющее скрыть внутренние IP-адреса
proxy - сервер-посредник, перенаправляющий запросы
bandwidth - пропускная способность сети, измеряемая в битах/с
latency - задержка передачи данных, измеряемая в миллисекундах
packet - пакет данных, передаемый по сети
socket - программный интерфейс для сетевого взаимодействия
port - номер порта, идентифицирующий приложение на устройстве
address - сетевой адрес устройства, например, IP-адрес
server - сервер, предоставляющий ресурсы или услуги другим устройствам
client - клиент, запрашивающий ресурсы или услуги у сервера
connection - соединение между устройствами для обмена данными
encryption - шифрование данных для обеспечения конфиденциальности
authentication - проверка подлинности пользователя или устройства
authorization - проверка прав доступа 
constructor - специальный метод, вызываемый при создании объекта
final - ключевое слово, указывающее на неизменяемость переменной или метода
static - ключевое слово, указывающее на принадлежность члена класса самому классу, а не его объектам
abstract - ключевое слово, определяющее абстрактный класс или метод
package - механизм организации классов в иерархическую структуру
import - оператор импорта классов из других пакетов
this - ссылка на текущий объект внутри класса
super - ссылка на родительский класс внутри класса-наследника
enum - перечисление, набор именованных констант
try-catch - конструкция для обработки исключений
finally - блок кода, выполняющийся всегда, независимо от возникновения исключений
multithreading - многопоточность, выполнение нескольких потоков одновременно
serialization - преобразование объекта в поток байтов для хранения или передачи
deserialization - обратный процесс преобразования потока байтов в объект
JDBC - API для работы с базами данных в Java
SQL - язык структурированных запросов для работы с базами данных
ORM - объектно-реляционное отображение, связывающее объекты Java с таблицами базы данных
JPA - спецификация для ORM в Java
REST - архитектурный стиль для создания веб-сервисов
JSON - формат обмена данными, часто используемый в REST API
XML - другой формат обмена данными, иногда используемый в веб-сервисах
SOAP - протокол для обмена структурированными сообщениями в веб-сервисах
WebSocket - протокол для двунаправленной связи между клиентом и сервером
RMI - удаленный вызов методов, механизм для вызова методов объектов на других машинах
JMS - Java Message Service, API для обмена сообщениями между приложениями
EJB - Enterprise JavaBeans, спецификация для компонентов бизнес-логики
Servlet - Java-класс, обрабатывающий запросы на веб-сервере
JSP - JavaServer Pages, технология для создания динамических веб-страниц
JSF - JavaServer Faces, фреймворк для разработки веб-приложений
Spring Boot - фреймворк, упрощающий создание Spring-приложений
Spring MVC - фреймворк для веб-приложений, основанный на Spring
Spring Data - фреймворк для работы с различными базами данных
Spring Security - фреймворк для обеспечения безопасности приложений
microservices - архитектурный стиль, разбивающий приложение на независимые сервисы
containerization - технология изоляции приложений с помощью контейнеров, например, Docker
cloud computing - облачные вычисления, предоставление вычислительных ресурсов по сети
AWS - Amazon Web Services, популярный провайдер облачных услуг
Azure - Microsoft Azure, другой популярный провайдер облачных услуг
GCP - Google Cloud Platform, еще один провайдер облачных услуг
Iaas - Infrastructure as a Service, облачная инфраструктура
PaaS - Platform as a Service, облачная платформа для разработки и развертывания приложений
SaaS - Software as a Service, облачное программное обеспечение
network topology - топология сети, описывает физическое или логическое соединение устройств
LAN - локальная сеть, объединяющая устройства в ограниченной области
WAN - глобальная сеть, объединяющая устройства на больших расстояниях
MAN - городская сеть, промежуточный вариант между LAN и WAN

routing - маршрутизация, процесс выбора пути для передачи данных
switching - коммутация, процесс пересылки данных между устройствами в одной сети
network security - сетевая безопасность, защита сети от угроз
encryption algorithms - алгоритмы шифрования, используемые для защиты данных
digital signature - цифровая подпись, механизм обеспечения целостности и аутентичности данных
firewall types - типы межсетевых экранов, различающиеся по принципам работы
intrusion detection system - система обнаружения вторжений, выявляющая подозрительную активность
